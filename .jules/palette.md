## 2024-05-24 - Visualizing CLI Mnemonics
**Learning:** Adding mnemonic keyboard shortcuts is only half the battle; users must be able to *discover* them. Highlighting shortcuts directly in menu items (e.g., `[S]tatus` instead of just `Status`) significantly improves discoverability without requiring a help manual.
**Action:** When adding shortcuts to CLI menus, always update the display string to visually indicate the trigger key.
